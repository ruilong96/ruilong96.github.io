Malaria Cell Image Detection

Ruilong Zhuang
Nov 13, 2020
Source: https://bit.ly/38O10lI

About the project

The main goal of this project is to detect the infected Malaria cells among the images of the parasitized and uninfected cells.

Methodology

  • Data: The dataset is acquired from Kaggle dataset a total of 27,558 cell images with equal instances of parasitized and uninfected cells, which are segmented cells from the thin blood smear slide images from the Malaria Screener research activity.

  • Metadata: The images are stored in two folders labelled as infected and uninfected. Each of images is in RGB format and has size of 142 × 163 pixels.

  • Since the dataset contains a set of image data and the project is a classification problem, I consider that convolution neural network(CNN) is the most reasonable and appropriate approach for this project. Also, it is because CNNs work well with data that has a spatial relationship.

  • Metrics: Tuned and evaluated models through computing the accuracy, presented by confusion matrix and accuracy score on the test set.

Result summary

  • Baseline model:
    • 94.4% accuracy on the test set
    • Precision = 0.9408122833085686
    • Recall = 0.9200775006054734

  • Regularized model:
    • 95.8% accuracy on the test set
    • Precision = 0.9501204819277108
    • Recall = 0.9661847586375888

  • Model using Keras:
    • 95.27% accuracy on the test set
    • Precision = 0.925703737886479
    • Recall = 0.9830923793187945
